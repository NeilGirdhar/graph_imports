=============
Graph Imports
=============

.. role:: bash(code)
    :language: bash

.. role:: python(code)
   :language: python

This repository allows one to graph the imports between packages.

-----------------------
Contribution guidelines
-----------------------

- Conventions: PEP8.

- How to clean the source:

  - :bash:`ruff .`
  - :bash:`pyright`
  - :bash:`mypy`
  - :bash:`isort .`
  - :bash:`pylint graph_imports`
